When അത് മൈനസ് ഇൻഫിനിറ്റി യിൽ നിന്ന് 0-ലേക്ക് പോകുന്നു എന്തു സംഭവിക്കും എന്ന് നിങ്ങൾ കരുതുന്നു- എന്ത് മൂല്യം ആണ് അത് മൈനസ് ഇൻഫിനിറ്റി മൈനസ് എന്തായാലും അത് 0ആയിരിക്കും . 0-ലേക്ക് പോകുമ്പോൾ, അത് പ്ലസ് അനന്തതയിലേക്ക് ഊതിവീർക്കും, കാരണം അത് നെഗറ്റീവ് സൈഡിൽ നിന്ന് വരുന്നു.

അപ്പോൾ, എന്താണ് സംഭവിക്കുക, അത് മൈനസ് റെഡി യിൽ നിന്ന് അത് പോകുന്നു തുടർന്ന് ഇത് 0 മൈനസ് ഒമേഗ ടെൻഡിംഗ് ആണ് . അതുകൊണ്ട്, ഞാൻ 0 പ്ലസ് ആൻഡ് 0 മൈനസ് ഇടാൻ പോകുന്നു, അത് ഏത് ഭാഗത്ത് നിന്നാണ് ഞാൻ 0 വരുന്നത് എന്ന് സൂചിപ്പിക്കാൻ മാത്രം. 0 പ്ലസ് എന്നാൽ ഞാൻ ശരിയായ ഓകെ നിന്ന് 0 പോകുന്നു എന്ന് അർത്ഥമാക്കുന്നു ; 0 മൈനസ് എന്നാൽ ഞാൻ നെഗറ്റീവ് യഥാർത്ഥ അച്ചുതണ്ടിൽ നിന്ന് ഇടത് നിന്ന് 0 വരുന്നു എന്നാണ് .

But തീർച്ചയായും ഇതൊരു സവിശേഷ ഉദാഹരണമാണ്, അതിനാൽ നിങ്ങളുടെ Nyquist plotഎന്ന നിലയിൽ അനന്തമായ ഒരു അർദ്ധവൃത്തത്തിന്റെ ആരം നിങ്ങൾക്ക് ഒരു കാര്യം പറയാൻ കഴിയും- എന്നാൽ ഇതൊരു സവിശേഷ ഉദാഹരണമാണ്. ഞാൻ നിങ്ങൾക്ക് ഒരു ഉദാഹരണം തരുന്നു-അത് എങ്ങനെ ശരിയായി ചെയ്യണമെന്ന് നിങ്ങൾക്ക് കാണിച്ചുതരാം.

അതിനാൽ, ഞങ്ങൾ നോക്കി ആദ്യം ഓർഡർ ഫാക്ടർ പ്ലോട്ട് ചെയ്യുക, തുടർന്ന് നിങ്ങൾക്ക് ഒമേഗ ഒരു പാരാമീറ്ററായി മൈനസ് ഇൻഫിനിറ്റി മുതൽ പ്ലസ് ഇൻഫിനിറ്റി വരെ വ്യത്യാസമുള്ള ഒരു പരാമീറ്റർ ആയി നിങ്ങൾക്ക് എന്ത് ലഭിക്കുമെന്ന് എന്നോട് പറയുക, അതിനാൽ ഞാൻ ഗൃഹപാഠം ഫൈൻ ആയി അത് നിങ്ങൾക്ക് വിടാൻ പോകുന്നു . അതിനാൽ, അടുത്ത ക്ലാസ്സിൽ നമ്മൾ കണ്ടുമുട്ടുമ്പോൾ, മാപ്പിങ്ങ് സൈദ്ധാന്തികം നോക്കുമ്പോൾ, തുടർന്ന് നാം നൈക്വിസ്റ്റ് സ്ഥിരതാ മാനദണ്ഡംചർച്ച ചെയ്യും പോലെ.
